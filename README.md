# Diego Bouzada — Portfolio

Sitio personal con mi CV interactivo y portfolio profesional.

**🔗 Demo en vivo:** [diegobouzada.vercel.app](https://diegobouzada.vercel.app)

---

## ✨ Sobre el sitio

Landing page bilingüe (Español / Inglés) que presenta mi perfil como Data & Analytics Engineer: experiencia profesional, stack técnico, servicios que ofrezco como freelance y vías de contacto.

- 🌎 Selector de idioma ES/EN
- 🎬 Animaciones con Framer Motion
- ✨ Fondo interactivo con partículas (tsparticles)
- 📊 Analytics integrado (Vercel Analytics)
- 📱 Totalmente responsive

---

## 🛠️ Stack técnico

- **Framework:** Next.js 16 (App Router) + React 19
- **Lenguaje:** TypeScript
- **Estilos:** Tailwind CSS
- **Animaciones:** Framer Motion, tsparticles
- **Iconos:** Lucide React, React Icons
- **Deploy:** Vercel

---

## 🚀 Correrlo en local

```bash
npm install
npm run dev
```

Abrí [http://localhost:3000](http://localhost:3000) para verlo.
